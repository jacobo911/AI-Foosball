A basic GUI for the AI-Foosball Capstone Project.

run the application in developer mode using:
`npm run start` 
in a command terminal. 

To bring up the developer tools:
`CTRL+SHIFT+I`

To compile, from the command line:
`npx electron-packager .`
The app will be packaged into `./foosball-gui-linux-armv7l`. Use `./foosball-gui-linux-armv7l/foosball-gui`
